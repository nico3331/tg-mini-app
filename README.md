🧮 TaxCalc — Калькулятор комиссии аукциона

Легкое и быстрое веб-приложение, разработанное специально для использования в качестве Telegram Mini App. Позволяет мгновенно рассчитать сумму лота после вычета комиссии аукциона и скопировать готовый результат.
✨ Ключевые возможности (Features)

    Интеграция с Telegram: Использует API telegram-web-app.js для нативной работы внутри мессенджера (приложение автоматически разворачивается на весь экран).

    Быстрый расчет: Вычисляет чистую сумму по формуле Сумма / 1.05 с точным математическим округлением.

    Умное копирование: Встроенная кнопка копирования в один клик. В интерфейсе число отображается красиво (с пробелами, например 100 000), но в буфер обмена копируется "чистое" число без пробелов (100000) для удобной вставки в другие программы.

    Интуитивный интерфейс: Минималистичный дизайн с поддержкой рублевой валюты (₽).

🛠 Технологии (Tech Stack)

    HTML5

    CSS3 (подключаемый файл styles.css)

    Vanilla JavaScript (логика расчетов script.js)

    Telegram Web Apps API

🚀 Как запустить и использовать

Так как проект состоит исключительно из фронтенд-файлов, его очень легко развернуть:

    Загрузите файлы (index.html, script.js и папку css) на любой бесплатный хостинг для статических сайтов (например, GitHub Pages, Vercel или Netlify).

    Скопируйте полученную ссылку на ваш сайт.

    Перейдите в Telegram к боту @BotFather.

    Настройте кнопку Web App для вашего бота, вставив скопированную ссылку.

Заметка по логике: Приложение принимает введенное значение, делит его на коэффициент 1.05 (расчет включенной комиссии) и выводит результат с локализацией ru-RU.
